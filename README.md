# DAB-Radio
In dit project zal een radio ontwikkeld worden die DAB+-signalen kan ontvangen en weergeven. De kern van de radio zal een SI4684 chip van SiliconLabs zijn. Er wordt een break-out-bord voorzien dat deze chip en de nodige randcomponenten bevat. Naast de radio-ontvanger moet een analoge versterker geïntegreerd worden die het mogelijk maakt om een 8Ω-luidspreker aan te sturen. Verder moet het mogelijk zijn om de radio in te stellen (zenderkeuze, volume, ...) via een smartphone gebruikmakend van Wifi of Bluetooth. De zenderkeuze en meta-informatie die in het radiosignaal vervat zitten, moeten via een scherm weergegeven worden. Voor het aansturen van het scherm en de radiochip zal een ESP32 microcontroller gebruikt worden. Deze microcontroller bevat een Wifi- en Bluetoothontvanger die voor de communicatie zal instaan. De microcontroller zelf is compatibel met Arduino zodat hij eenvoudig te programmeren is.
